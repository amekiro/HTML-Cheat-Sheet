Update from HTML4.01 to HTML5

What you should know when writing new HTML5 files

With HTML5, new semantic elements were created. This makes content easier to read in the HTML file removes the need to label div elements with ids or classes. Below is a list of new semantic elements that can be used.

Tag Name Structure Replaces Example
article <article> <!-- Insert your content here --> </article> <div class="item"> <!-- Insert your content here --> </div> <article> <h4>This is my section title</h4> <p>Let's write some content here!</p> </article>
